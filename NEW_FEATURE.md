# 新增功能说明：扫码历史记录

## 功能描述

在扫描结果页增加历史记录保存功能，用户每次扫码成功后自动保存结果。同时新增“扫码历史”页面，支持查看、清空所有历史扫码记录。

## 新增/修改文件

### 新增文件

| 文件 | 功能 |
| :--- | :--- |
| `common/model/ScanRecord.ets` | 扫码记录数据模型 |
| `pages/scanHistory/ScanHistory.ets` | 扫码历史记录列表页 |

### 修改文件

| 文件 | 修改内容 |
| :--- | :--- |
| `pages/Index.ets` | 新增“扫码历史”入口按钮 |
| `pages/resultPage/ResultPage.ets` | 扫码成功后自动保存记录 |
| `main_pages.json` | 注册新页面路由 |

## 技术要点

1. 使用 **Preferences** 进行数据持久化存储
2. 使用 **@State** 实现状态管理和 UI 刷新
3. 使用 **List + ForEach** 实现列表展示
4. 处理空状态 UI 展示

## 运行效果

1. 扫码成功后自动保存记录
2. 首页点击“扫码历史”进入记录列表
3. 列表展示扫码内容和时间
4. 支持一键清空历史记录
